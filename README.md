# 🚀 45 Days Data Engineering & Python Interview Challenge

Welcome to my **45 Days Challenge**! This repository documents my journey to strengthen my skills in **Python**, **SQL**, **ETL**, **PySpark**, **Databricks**, **AWS**, and **Data Engineering** through daily learning, coding, projects, and interview preparation.

The goal is to build production-ready skills while maintaining consistency by learning and coding every day.
